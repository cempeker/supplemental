

##### This file provides information on the contents of supplementary material.


1. "Data" folder includes our experimental data. More information on the data and variables can be found in /data/variables.txt

2. "Experiment instructions" provides the pdf exports of the instructions part in each experimental condition.

3. The R scripts "analysis_s1.R" and "analysis_s2.R" reproduces the figures, tables and statistical results in Study 1 and 2 respectively. The scripts can be run using R version 3.6.3 if the R libraries used in the script are installed (freely available for download through various CRAN mirrors)
